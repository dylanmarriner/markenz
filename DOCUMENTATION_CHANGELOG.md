# Documentation Changelog

All notable changes to the Markenz documentation will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Docs-as-code](https://www.writethedocs.org/guide/docs-as-code/) principles.

## [2.0.0] - 2026-01-19

### Added

- **Executive Overview:** High-level summary for non-technical stakeholders (`EXECUTIVE_OVERVIEW.md`).
- **ADR System:** Introduced Architecture Decision Records in `docs/adr/`.
- **Maturity Model:** Added operational maturity definition in `docs/v1/MATURITY_MODEL.md`.
- **Roadmap:** Formalized execution roadmap in `docs/v1/ROADMAP.md`.
- **Diagrams System:** Established `docs/diagrams/source` and `docs/diagrams/rendered` for editable visual artifacts.

### Changed

- **Directory Structure:** Migrated flat `docs/` structure to versioned `docs/v1/` hierarchy.
- **README.md:** Rewritten for executive clarity and enterprise standard navigation.
- **Tone:** Enforced authoritative, non-conversational voice across all artifacts.

### Removed

- **Legacy Audits:** Archived outdated audit reports to `docs/archived/` to reduce noise in the active documentation set.
